TodoTracker.js

Display your productivity over time

TodoTracker lets you add tasks to a to-do list and track your productivity over time, in a way similar to the GitHub contribution graph.

A darker colored square means more tasks completed on that day!

Examples

This is the basic interface for a TodoTracker.

Today is , so any tasks you tick off will update that day's square.

Code 
const todo = new TodoTracker()
todo.addAfter('#ex1')

With default settings, the graph spans over 4 months. But you can easily change the range & the starting month!

Here's a year-long one, filled out for display purposes.

Code 
const todo_year = new TodoTracker({random: true, time: 12})
todo_year.addAfter('#ex2')

I've added a dropdown under this one to simulate adding tasks on different days.

Try it out! Choose different dates and tick off tasks to see the graph get filled.

Code 
const todo_choose = new TodoTracker({customDate: true, start: 10})
todo_choose.addAfter('#ex3')

Themes

TodoTracker has different color themes to choose from.

Check them out! Click on the tabs on the side to try the current themes.

  • Blue
  • Green
  • Orange
  • Pink
  • Gray
Code 
const todo_color = new TodoTracker({random: true, color: 'blue'})
todo_color.appendTo('.themes-cont')
// Can also call a method to change color
todo_color.changeColor('green')
