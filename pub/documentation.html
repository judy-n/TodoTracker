Getting Started

First, download the files TodoTracker.js and TodoTracker.css and import them in your HTML file. 

<script defer src="todotracker.js"></script> 
<link rel="stylesheet" href="todotracker.css">

Create a TodoTracker object in your JavaScript file. With no parameters, the default will span over 4 months starting from the current one, and have a blue theme.

const todo = new TodoTracker()

Then you can add it before, after, or append it to any container in the DOM.

todo.addAfter('.title')

API

A TodoTracker object takes an optional parameter: an options object.

const todo = new TodoTracker(options)

options takes the properties below. Passing in any property will replace it, otherwise it keeps the default value.

options

Property Default Type Description
start The current month Int The month number (1-12) that the productivity graph starts from.
time 4 Int The length (max. value is 12) that the graph spans over.
color 'blue' String The color theme. You can choose between:
  • 'blue'
  • 'green'
  • 'orange'
  • 'pink'
  • 'grey'
random false Boolean Setting this to true will fill the graph squares with randomly distributed shades. Mainly used for displaying purposes or a proof of concept.
customDate false Boolean Setting this to true adds a dropdown menu that lets you choose any date and tick off tasks for it.d

Methods

Below are the methods that can be called on a TodoTracker object.

Method Params Description
appendTo query: String
Selector for querySelector		 Appends the TodoTracker object to the queried container in the DOM.
addBefore query: String
Selector for querySelector		 Places the TodoTracker object in the DOM before the queried container.
addAfter query: String
Selector for querySelector		 Places the TodoTracker object in the DOM after the queried container.
changeColor color: String
Has to be one of:
  • 'blue'
  • 'green'
  • 'orange'
  • 'pink'
  • 'grey'
 Changes the color theme of the TodoTracker to the chosen color.
